Data Science Project

A project exploring the effect on pay on investment banks' performance

This first chart shows the performance of the top 10 investment banks globally, based on total fee value for M&A alone. This provides a snapshot of the major players in the sector - it is these banks whose pay will be compared to their performance. The chart can be recreated here.
This section outlines the pay landscape for Investment Bankers in London
Arkesden is a head hunter, based in London, which recruits for Private Equity, Real Estate, Alternatives, Energy and (crucially), Investment Banking. Each year they produce a report which summaries the pay in each sector, which is a free-to-download PDF which summarises the pay in each sector for each bank. A PDF scraper has been built to scrape this PDF, meaning that each year when the report is published, this website can easily be scraped. The link for the scraper is here.
The above charts summarise the landscape, and compare how banks have paid in 2020/2021 depending on the position. From immediate inspection, it is clear that Bank of Merrill Lynch (BAML) pays more than other banks, especially at the Analyst rank (the lowest level). This project will then look and see if this translates to greater performance overall.
The above chart shows the total compensation, including both base pay and bonus pay, the latter often being larger than the former. Bonus is a function of the performance of the individual and so using this data will also allow us to analyse whether performance related pay (PRP), on a bank level, also affects performance. The data from the chart was again scraped from Arkesden and can be replicated here.
